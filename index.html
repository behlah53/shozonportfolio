About Me

Hi, I'm Shozon Roy, a Frontend Developer, UI/UX Designer, and Tech Blogger from Dinajpur, Bangladesh. I have a deep passion for creating beautiful and functional user interfaces, specializing in frontend development and UI/UX design.

With over 100+ web projects built, I have extensive experience in HTML, CSS, JavaScript, Bootstrap, jQuery, and Tailwind CSS. While I have some backend knowledge in PHP and Python, my main focus is on crafting modern, user-friendly, and interactive web experiences.

Beyond coding and design, I run a tech blog where I have published 40+ articles on web technologies, UI/UX trends, and industry insights. I love sharing my knowledge and helping others learn about frontend development and design.
